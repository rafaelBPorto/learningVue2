Para se utilizar uma propriedade dentro de uma tag HTML é preciso utilizar diretivas

Diretivas são propriedades personalizadas que serão intepretadar pelo VUE, pode se criar suas propias diretivas ou utilizas as fornecidas pelo Vue.

Um exemplo de diretita é: v-bind


        <a v-bind:href="link">s</a>
        
        <script >
          new Vue({
            data: {
              link: "http://google.com",
            }
          });
        </script>

Resultado:

Google
